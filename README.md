# rkgit-hackwarts-StudyCanvas
**Theme:** Smart Education | Hackwarts 2026 @ RKGIT


## About
StudyCanvas is a powerful, lightweight digital notebook that replaces physical books, copies, and pens. It delivers a true **bagless campus experience** for B.Tech students with smooth drawing, multi-page support, PDF export, and smart organization.


## Tech stack
- HTML
- CSS
- JavaScript (Vanilla JS)
- Canvas API
- IndexedDB + LocalStorage (offline-first)


## Live demo
Live Prototype Deployed on Vercel: (https://rkgit-hackwarts-study-canvas-eta.vercel.app/)


**Fully responsive • 100% offline • Advanced Canvas Editor**


## Key Features (50+ capabilities across 5 pages)
- **10 Drawing Tools**: Bézier Pen, Highlighter (35% opacity), Eraser, Text, Line, Arrow, Rectangle, Circle/Ellipse, Triangle, Select & Move with floating ImageData

- **Advanced Canvas Controls**: Zoom 25–500%, 50-step undo/redo, grid overlay, clear page, multi-page support, board background toggle

- **Smart Organization**: Lazy-loaded note grid, real-time search, subject filter chips & badges, rename/delete, inline title editing

- **Productivity Tools**: Auto-save every stroke, storage status bar, PDF export of all pages, JSON backup & restore

- **Settings**: Light/Dark/System theme, text size slider (12–20px), default pen width, clear all data, app version & about

- **Offline-First Architecture**: IndexedDB storage, transparent PNG pages, localStorage migration, 100% offline


## Problem it Solves
- Heavy bags and back pain
- Lost or damaged paper notes
- Paper waste
- Temporary NEP 2020 Bagless Days — we made it permanent for college
